# 分类
| 名称     | 描述 |
| -------- | ---- |
| 内部总线 |微机内部各个外围芯片，与处理器之间的总线。芯片级别      |
| 系统总线 |微机中各个插件板，与系统板之间的总线，如PCI、VGA接口。插件级别      |
| 外部总线 |微机与外部设备之间的总线      |

## 系统总线
| 名称     | 描述 |
| -------- | ---- |
| 数据总线 | 代表能传输的位数，64位计算机则最高传输64bit |
| 地址总线 | 地址是32位，则地址空间是2的32次方=4G宽度；能装64位系统，是因为硬件支持，也就是支持地址存储64位 |
| 控制总线 |微机与外部设备之间的总线      |
