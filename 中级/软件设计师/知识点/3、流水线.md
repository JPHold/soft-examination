# 概念
> 多条指令交替执行的**准并行**处理技术
> 为何是准并行：并不是同时执行多条指令，而是在某个时刻，a指令执行到b1步骤、b指令执行执行a1步骤，假如a指令一直塞在b1步骤，那么b指令只能等待（**所以其实是串行执行**）
